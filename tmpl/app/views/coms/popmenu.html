Popmenu组件

该组件只提供功能，不提供界面，主要关联触发节点与要显示的节点

配置项

参数名称 参数类型 参数说明 默认值
related string 关联的节点id，包含#号
placement string 关联节点在触发节点的方位，仅top与bottom bottom
align string 关联节点与触发节点的对齐方式，仅left与right left
