下拉框组件

配置项

参数名称 参数类型 参数说明 默认值
list array 列表数组，可为简单数组或[1,2,3]或对象数组，如[{id:1,text:'a'}]
textKey string 当列表中的项为对象时，渲染text时读取的key
valueKey string 当列表中的项为对象时，渲染value时读取的key
searchbox boolean 是否包含搜索框 false
disabled boolean 是否禁用 false

事件

change
参数名称 参数类型 参数说明 默认值
item object 选中对象
keys object 包含textKey与valueKey的对象
value string 选中值
text string 选中文本
